# Project Instructions

- Branch names must always be in English.
- Commit messages must always be in English and use Conventional Commits.
- This app is already in production with real user data.
- There is currently no backup available for existing production data.
- Any change must preserve existing data and avoid destructive operations, unsafe migrations, data wipes, storage resets, or storage replacement without an explicit data-preservation strategy.
- Treat data preservation as a priority constraint for all future work involving code, database/storage, migrations, sync, or deployment.
